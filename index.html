Git & GitHub

Before knowing what Git is, any beginner should have an idea of a Version Control System, or VCS for short. A Version control system, helps a developer or an organisation keep track of the changes they make to their products over time. This is done by saving different versions of their code and data by a process known as versioning. It also allows multiple developers or employees of an organisation work on the same section of software at the same time by "branching" the work.

Git is a VCS created and maintained by the developers of Linux operating system, for the necessity to have a VCS to version the operating system itself. Git, unlike previous VCS's has its own method of version control. Git repositories are initialized in a working directory to initiate the versioning process. Commits are made to the repository whenever instances are desired to be saved. Git, unlike other VCS's does not "delta compress" files while a commit is made. Delta compression is a method of storing changes made to a file in the form of additions and deletions. At the time of commits, Git treats the files as a single 'snapshot' and stores all the data as a modified file.

Git also helps developers to share data by "pushing" and "pulling" repositories. Git does not work on any server. When a pull is initiated, Git creates an exact copy of the original repository in the user's local computer. A user can work offline with Git. Other VCS's lack such features.

Eclipse IDE & Tools

Eclipse is more than just an Integrated Development Environment (IDE); it is a community of developers. This means that Eclipse is under a state of constant and fast development.

An IDE is a workstation which a developer uses to develop applications in an easy way. The benefit of an IDE is that it automates some basic development processes like creating makefiles for C++ development, xml web descriptors for web development, compilation of source code, etc. This is very helpful for the programmer, who does not have to go through that procedure for every project.

The best part of eclipse is that it is based on the idea of plugins. Almost any kind of development in any language is made possible with a suitable plugin pack for the IDE. Many features of Eclipse help in providing easy and quick development experience, and neat maintenance of the working directory structure.

Some Popular languages of today

Python

Python is a wonderful programming language. I say so because its syntax is very simple and intuitive, unlike C or C++. You do not need to mess with the curly braces and the semicolons anymore. Python has an extensive library support for developing almost anything.

It has library methods for scripting Operating System Scripts to developing web applications. One drawback with python is that it is a slow language in execution process. You wouldn't use it for developing or maintaining operating systems.
