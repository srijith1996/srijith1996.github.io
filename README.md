# My GitHub IO page

This webpage is a simple one involving no server-side tasks.
It consists of some client-side utilities like a calculator, the javascript for which is not quite difficult. This repository is largely for my learning purpose and does not contain any specific stuff for the community accept those like me trying to learn programming and some tools.
For this purpose there is a knowledge section which has drop-down list for details on useful sources for learning.
The calculator has two parts. A simple one which is already deployed, and an advanced one which is still on the working.
The webpage does not have viewport issues and is scalable on any device.

The webpage is fully operational at srijith1996.github.io
